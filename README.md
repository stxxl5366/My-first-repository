## 第七个标签：列表
无序列表可以使用星号，加号或者减号（*、+、-）创建，
有序列表则使用数字接着一个英文句点：
1.  Bird
2.  McHale
3.  Parish

## 第八个标签：分割线
要添加分割线非常简单，直接在一个空行里打上三个星号(***)。
